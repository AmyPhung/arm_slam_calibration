## Setup
+ `cd ~/catkin_ws/src`
+ `git clone https://github.com/AmyPhung/arm_slam_calibration`
+ `cd ~/catkin_ws/`
+ `rosdep install -iry --from-paths src`


## Usage
+ Start apriltag detection `roslaunch apriltag_ros continuous_detection.launch`
